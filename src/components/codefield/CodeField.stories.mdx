### Usage:

To use, import the component `CodeField` from `@nilfoundation/ui-kit`.

<Source
  language="tsx"
  dark
  format={true}
  code={`
        import {CodeField} from "@nilfoundation/ui-kit";
        ...
        <CodeField
          code="console.log("Hello World")"
          displayCopy={false}
          onBeforeCopy={str => "Copied message: " + str}
          onCopy={() => console.log("Copied!")}
        />
  `}
/>

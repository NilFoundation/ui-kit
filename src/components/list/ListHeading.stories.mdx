### Usage:

To use, import the component `ListHeading` from `@nilfoundation/ui-kit`.

<Source
  language='tsx'
  dark
  format={true}
  code={`
        import {ListHeading, Button} from "@nilfoundation/ui-kit";
        ...
        <ListHeading heading="Heading" subHeading="Sub-heading" />
        // With end enhancer
        <ListHeading heading="Heading" subHeading="Sub-heading" endEnhancer={() => <Button>Action</Button>} />
  `}
/>

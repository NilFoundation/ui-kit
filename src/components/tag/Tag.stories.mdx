### Usage:

To use, import the component `Tag` from `@nilfoundation/ui-kit`.

<Source
  language="tsx"
  dark
  format={true}
  code={`
        import {Tag, TAG_KIND, TAG_SIZE} from "@nilfoundation/ui-kit";
        ...
         <Tag>Tag label</Tag>
        // Disabled
         <Tag>Tag label</Tag>
        // With kind

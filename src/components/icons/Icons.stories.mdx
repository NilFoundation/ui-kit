### Usage:

To use, import the icon component from `@nilfoundation/ui-kit`.

<Source
  language='tsx'
  dark
  format={true}
  code={`
        import {SearchIcon} from "@nilfoundation/ui-kit";
        ...
        <SearchIcon />
        // With color
        <SearchIcon color="#000" />
        // With size
        <SearchIcon size={32} />
  `}
/>

### Usage:

To use, import the component `Spinner` from `@nilfoundation/ui-kit`.

<Source
  language="tsx"
  dark
  format={true}
  code={`
        import {Spinner, SPINNER_SIZE} from "@nilfoundation/ui-kit";
        ...
        <Spinner />
        // With size
        <Spinner size={SPINNER_SIZE.large} />
        // With animation
        <Spinner animation />
        // With label
        <Spinner animation label="Uploading..." />
  `}
/>
